Política de Privacidade

Última atualização:

1. Coleta de Dados

Coletamos dados como nome, telefone, informações trabalhistas e demais dados necessários para a prestação dos serviços oferecidos pela empresa.

2. Uso das Informações

Os dados coletados são utilizados para:

  • Disponibilização de holerites
  • Gestão de ponto eletrônico
  • Assinatura de documentos internos
  • Envio de notificações via WhatsApp Business

3. Uso do WhatsApp Business

Utilizamos a API oficial do WhatsApp Business fornecida pela Meta Platforms, Inc. para envio de notificações transacionais aos colaboradores. Nenhuma mensagem promocional é enviada sem consentimento.

4. Compartilhamento de Dados

Não compartilhamos dados pessoais com terceiros, exceto quando necessário para cumprimento de obrigação legal ou determinação judicial.

5. Segurança

Adotamos medidas técnicas e organizacionais para proteger os dados contra acessos não autorizados.

6. Direitos do Usuário

O titular dos dados pode solicitar acesso, correção ou exclusão de seus dados através do contato oficial da empresa.

7. Contato

Para dúvidas sobre esta política, entre em contato com a empresa responsável.
